# 簡単な解決方法

PostgreSQLのパスワードがわからない場合、以下の2つの選択肢があります：

## 選択肢1: PostgreSQLのパスワードをリセット（推奨）

上記の「パスワードリセット手順_手動.md」を参照してください。

## 選択肢2: SQLiteを使用（最も簡単）

PostgreSQLのパスワードをリセットするのが面倒な場合、SQLiteを使用できます。

### 必要なもの
- Visual Studio Build Tools（無料、10-20分でインストール）

### 手順

1. **Visual Studio Build Toolsをインストール**
   - https://visualstudio.microsoft.com/downloads/ にアクセス
   - 「Build Tools for Visual Studio」をダウンロード
   - インストール時に「C++によるデスクトップ開発」を選択
   - インストール後、PowerShellを再起動

2. **環境変数をSQLite用に設定**
   ```powershell
   cd C:\Users\ngf005148\Desktop\webapps\Pic_cul\backend
   notepad .env
   ```
   
   以下の内容に変更：
   ```env
   DB_TYPE=sqlite
   DATABASE_URL=./data/pic_cul.db
   STORAGE_TYPE=local
   PORT=3001
   JWT_SECRET=dev-secret-key-change-this-in-production
   NODE_ENV=development
   ```

3. **依存関係をインストール**
   ```powershell
   npm install
   ```

4. **マイグレーション実行**
   ```powershell
   npm run db:migrate
   npm run db:seed
   ```

5. **開発サーバー起動**
   ```powershell
   cd ..
   npm run dev
   ```

## 推奨

**今すぐ開発を始めたい場合**: 選択肢2（SQLite）が最も簡単です。

**本番環境と同じ環境で開発したい場合**: 選択肢1（PostgreSQLパスワードリセット）を推奨します。
