## ⚠️ 必要な作業（手動）

### SQLiteを使用する場合

SQLiteを使用するには、**Visual Studio Build Tools**のインストールが必要です。

#### インストール手順

1. **ブラウザで以下にアクセス**
   ```
   https://visualstudio.microsoft.com/downloads/
   ```

2. **ページを下にスクロール**
   - 「Tools for Visual Studio」セクションまでスクロール

3. **「Build Tools for Visual Studio」をクリック**
   - ダウンロードが開始されます

4. **インストーラーを実行**
   - ダウンロードした `vs_buildtools.exe` を実行

5. **ワークロードを選択**
   - 「**C++によるデスクトップ開発**」にチェックを入れる
   - 右側で必要なコンポーネントが自動選択されます

6. **インストール**
   - 「インストール」ボタンをクリック
   - インストールには10-20分かかります

7. **PowerShellを再起動**
   - インストール後、**PowerShellを完全に閉じて再度開いてください**

8. **依存関係をインストール**
   ```powershell
   npm run db:migrate
   npm run db:seed
   ```

9. **マイグレーション実行**
   ```powershell
   npm run db:migrate
   npm run db:seed
   ```

10. **開発サーバー起動**
    ```powershell
    cd ..
    npm run dev
    ```

### PostgreSQLを使用する場合

PostgreSQLサービスを起動する必要があります。

1. **Windowsキー + R** を押す
2. `services.msc` と入力してEnter
3. サービス一覧で「**postgresql**」で始まるサービスを探す
4. 右クリック → 「**開始**」を選択

サービスが起動したら、データベースを作成できます。
